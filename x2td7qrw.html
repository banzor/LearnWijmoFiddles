		values by overriding the <b>GroupDescription</b>'s
		<b>groupNameFromItem</b> method.</p>

		value bins (e.g. high, low, medium), date ranges
		(e.g. this week, this year), etc.</p>

		<b>GroupDescription</b> that returns the
