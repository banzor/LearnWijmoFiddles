  <h1>
    Wijmo Olap Architecture</h1>
 <p>
    Pivot tables are one of Excel's most powerful features. They
    allow you to explore large, detailed data sets and detect
    important trends and relations.
  </p>
  <p>
    Wijmo's <b>wijmo.olap</b> module allows you to add this 
    functionality to your JavaScript applications.
  </p>
  <p>
    The diagram below shows the relationships between the 
    four main classes in the <b>wijmo.olap</b> module:
  </p>

  <div class="diagram" aria-label="Wijmo Olap Architecture Diagram">
    <div class="raw-data-client">
        <h4>
          Raw Data (client-side)
        </h4>
        <p>
          Arrays with plain JavaScript objects.
        </p>
    </div>
    <div class="raw-data-server">
        <h4>
          Raw Data (server-side)
        </h4>
        <p>
          Connection to OLAP SASS cubes or
          ComponentOne Data Engine services.
        </p>
    </div>
    <div class="pivot-engine component">
        <h4>
          PivotEngine
        </h4>
        <p>
          Summarizes the raw data according to views
          defined by fields and field lists
          (row, column, filter, and value lists).
        </p>
    </div>
    <div class="pivot-panel component">
        <h4>
          PivotPanel
        </h4>
        <p>
          Provides a drag-drop UI for editing the
          <b>PivotEngine</b>'s fields and field lists.
        </p>
    </div>
    <div class="pivot-grid component">
        <h4>
          PivotGrid
        </h4>
        <p>
          Displays data summaries generated by a
          <b>PivotEngine</b> as tables with 
          collapsible rows/columns and detail
          drill-downs.
        </p>
    </div>
    <div class="pivot-chart component">
        <h4>
          PivotChart
        </h4>
        <p>
          Displays data summaries generated by a
          <b>PivotEngine</b> as charts with 
          hierarchical axes.
        </p>
    </div>

    <!-- connectors -->

  <p>
  	The <b>PivotEngine</b> component is the core component in 
    the <b>wijmo.olap</b> module. It takes raw data and creates
    summary views based on field lists (there are value,
    row, column, and filter lists). The server can use local data
    or connect to servers such as OLAP SASS cubes.
  </p>
  <p>
    The <b>PivotPanel</b> control allows users to see and edit view 
    definitions being used by a <b>PivotEngine</b>.
    The control shows a list of fields that can be dragged to row, 
    column, value, or filter areas, like in Excel.
    When users modify the view definition, the engine automatically
    updates the output data summary.
  </p>    
  <p>
  	View definitions can also be created and modified by using the
    <b>PivotEngine</b>'s object model directly.
  </p>
  <p>
    The <b>PivotGrid</b> and <b>PivotChart</b> controls display the
    output data summaries generated by the <b>PivotEngine</b>.
    Both controls allow you to export the results to PDF or XLSX.
  </p>    

</div>
