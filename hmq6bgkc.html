<div class="container">

  <h1>
    Other Classes and Services
  </h1>
  <p>
    In addition to implementing fundamental classes such as
    <a href="https://wijmo.com/5/docs/topic/wijmo.Control.Class.html" target="_blank">Control</b>,
    <a href="https://wijmo.com/5/docs/topic/wijmo.Event.Class.html" target="_blank">Event</a>, and 
    <a href="https://wijmo.com/5/docs/topic/wijmo.collections.CollectionView.Class.html" target="_blank">CollectionView</a> classes, the <b>wijmo.js</b>
    module provides many other useful services and classes including:</p>
  <ul>
    <li>
      <a href="https://wijmo.com/5/docs/topic/wijmo.Tooltip.Class.html" target="_blank">Tooltip</a>:
      Class that provides pop-up windows that display additional information about elements on the page.</li>
    <li>
      <a href="https://wijmo.com/5/docs/topic/wijmo.Clipboard.Class.html" target="_blank">Clipboard</a>:
      Class that provides static <b>copy</b> and <b>paste</b> methods that can be used by applications 
      and controls to customize the clipboard content during clipboard operations.</li>
    <li>
      <a href="https://wijmo.com/5/docs/topic/wijmo.PrintDocument.Class.html" target="_blank">PrintDocument</a>:
      Class that makes it easy to create documents for printing or exporting to PDF.
      Most browsers allow you to select the paper size, orientation, margins, 
      and whether to include page headers and footers.
      </li>
    <li>
      <a href="https://wijmo.com/5/docs/topic/wijmo.Color.Class.html" target="_blank">Color</a>:
      Class that parses colors specified as CSS strings and exposes their red, green, blue,
      and alpha channels as read-write properties. Also provides methods for converting
      color spaces and interpolating.</li>
    <li>
      <a href="https://wijmo.com/5/docs/static/glyphs.html" target="_blank">Glyphs</a>:
      The wijmo.css file includes several glyphs used by the Wijmo controls. These glyphs
      do not depend on any font files or external images, and you may use them in your
      applications as well.</li>
    <li>
      <a href="https://wijmo.com/5/docs/topic/wijmo.Module.html#httpRequest" target="_blank">httpRequest</a>:
      Static method used to submit http requests to the server.</li>
    <li>
      <a href="https://wijmo.com/5/docs/topic/wijmo.Module.html#animate" target="_blank">animate</a>:
      Static method used to create animations by setting properties on a timer.</li>
  </ul>
