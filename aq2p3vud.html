<div class="container">

  <h1>
    Chart Events
  </h1>
  <p>
    The FlexChart provides events you can use to 
    customize the appearance and behavior of the
    chart.</p>
  <p>
    These events can be divided into categories:</p>
  <ul>
    <li>
      <b>Mouse</b>:
      The FlexChart relies on HTML events for mouse handling,
      and provides a <b>hitTest</b> method that can be used
      to determine which part of the chart the event refers to.
      Mouse events can be used to add functionality such as
      zooming or drill-down.</li>
    <li>
      <b>Selection</b>:
      When the chart's <b>selectionMode</b> property is set 
      to <b>Series</b> or <b>Point</b>, the chart will allow users
      select elements by clicking on them. The chart's <b>selection</b>
      property will be updated and the <b>selectionChanged</b> event
      will be fired. This event is typically used in master/detail
      scenarios.</li>
    <li>
      <b>Rendering</b>: 
      The FlexChart has <b>rendering</b> and <b>rendered</b> events
      that allow you do add custom elements to the chart, either in
      the background or as overlays.</li>
  </ul>  
</div>
