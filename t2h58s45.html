  <h1>
    Referencing Wijmo 5 in your Applications</h1>
  <p>
    To use Wijmo 5 in your applications, include it by adding a
    few references to your HTML pages.</p>
  <p>
    The minimal set of files required by any Wijmo 5 application is:</p>
  <ul>
    <li><b>wijmo.css</b>: Contains the CSS rules used for styling all 
      Wijmo 5 controls.</li>
    <li><b>wijmo.js</b>: Contains the Wijmo 5 infrastructure including
      the Globalize, Event, Control, and CollectionView classes.</li>
  </ul>
  <p>
    In addition to these, include one or more additional files,
    depending on which components you use:</p>
  <ul>
    <li><b>wijmo.grid.js</b>: Contains the FlexGrid control.</li>
    <li><b>wijmo.chart.js</b>: Contains the FlexChart and FlexPie controls.</li>
    <li><b>wijmo.input.js</b>: Contains several input controls, including ComboBox,
      AutoComplete, InputDate, InputTime, InputNumber, InputMask, ListBox, Menu,
      and Calendar controls.</li>
    <li><b>wijmo.gauge.js</b>: Contains several gauge controls, including 
      LinearGauge, RadialGauge, and BulletGraph.</li>
    <li><b>wijmo.culture.[CultureName].js</b>: Contains culture-specific files
      used to develop applications in languages other than American English.</li>
    <li><b>wijmo.theme.[ThemeName].css</b>: Contains CSS rules used to customize
      the appearance of the Wijmo 5 controls.</li>
    <li>Any other modules you need. Find a 
    <a href="https://wijmo.com/5/docs/">complete list here</a>.</li>
  </ul>
  
  <p>
    You may download Wijmo from our site and copy the required files to the 
    appropriate folders within your application, or you may reference Wijmo
    files from our CDN.
    <a href="https://wijmo.com/5/docs/static/references.html">
    See examples here</a>.</p>
</div>
