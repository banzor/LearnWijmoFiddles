        <b>ComboBox</b>: Combines an input element with a drop-down
        list and provides as-you-type search, making it quick and 
        easy to find items in long lists. Use it to enter strings
        or to select objects from lists.</p>
      <p>
        For example, select a country: <input id="theCombo"></p>
    </li>
    <li>
      <p>
        <b>AutoComplete</b>: Extends the ComboBox to provide
        as-you-type async loading of items and a customizable
        search algorithm. Use it to select items from large 
        datasets on the server.</p>
      <p>
        For example, type 'land' or 'uni': <input id="theAutoComplete"></p>
    </li>
    <li>
      <p>
        <b>InputMask</b>: Provides character-level validation 
        by allowing you to specify a mask composed of templates
        that constrain each character typed by the user.</p>
      <p>
        For example, type a phone number: <input id="theInputMask"></p>
    </li>
  </ul>
