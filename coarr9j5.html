<div class="container">

  <h1>
    AutoComplete
  </h1>

  <p>
    The AutoComplete control extends the ComboBox to provide
    two additional features:</p>
  <ol>
    <li>
      It automatically filters the items in the drop-down list 
      to include only those that match the current user input, and</li>  
    <li>
      It supports populating the drop-down asynchromously based
      on the current user input and changing the search logic
      (by default, the ComboBox looks for items that <b>start</b>
      with the user input, while AutoComplete looks for items that
      <b>contain</b> the user input).</li>  
  </ol>
  <p>
    The AutoComplete and ComboBox controls below are bound to the
    same <b>itemsSource</b>. Try typing "un" into either control
    to see the difference:
  </p>
  <label for="theAutoComplete">AutoComplete:</label>
  <div id="theAutoComplete"></div>
  <br/>
  <label for="theCombo">ComboBox:</label>
  <div id="theCombo"></div>
</div>
