  <p>
    Tabs are presented as a single row above their associated content.

    Tabs can be selected with the mouse or keyboard, and automatically
    update the content to reflect the current selection.</p>
  <p>
    Each tab consists of a header element (always visible) and a 
    pane element that contains the tab content. The pane element
    is visible when the corresponding header is selected with the
